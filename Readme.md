# Online Movie Ticket Booking System Backend

# technology used

1. Node.js
2. MySQL
3. Express.js

## Endpoints

- Booking a Seat
 ```bash
 api/v1/booking/create/:userId/:showId/:seatId
 ``` 
 
- Cancel the Booking

```bash
 api/v1/booking/cancel/:BookingId
 ``` 

- Get All Available Seats for a show

```bash
 api/v1/seat/available/:showId
 ```
